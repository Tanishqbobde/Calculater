## ✨ Features

- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
- Clear (AC) functionality
- Delete (DEL) functionality
- Responsive design
- Clean and intuitive user interface
- Real-time display updates

## 📚 What I Learned

While building this project, I gained a deeper understanding of:

- DOM Manipulation
- Event Handling
- JavaScript Functions
- Query Selectors
- Dynamic UI Updates
- Problem Solving & Debugging

The biggest challenge was understanding how JavaScript interacts with HTML elements through the DOM. Initially, connecting button clicks to calculator logic was confusing. By practicing DOM selection methods, event listeners, and updating elements dynamically, I was able to overcome the challenge and complete the project successfully.
